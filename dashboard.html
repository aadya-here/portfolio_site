about project


this is a dashboard created for a new pizzeria using data generated by me. i couldn't find anything online that was good enough so i just made it using basic python.
essentially the data was converted to a sql schema with 3 tables - orders, menu, customers. using this i have created 2 dashboards using sql commands in tableau.

implementation


before any analysis, i needed data. i found a random name generator online, and then used python to randomly generate data for the 3 csv files. the data was then uploaded on mysql work bench as per the schema so that it could be analyzed further in tableau.

challenges


  • creating usable data

  • creating a functional schema so that the data is stored and accessed efficiently

  • effectively visualising all the various data into a dashboard
