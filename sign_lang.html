about project


individuals with hearing and speaking issues face communication barriers, both in real life and even more so during digital interactions like video calls. present solutions offer real-time captions but fall short for those who rely on sign language.

tech stack


  • tensorflow (keras)

  • open cv

  • mediapipe library

implementation


open cv works on static images and hence is not fully suitable for this purpose. google's media pipe library is the perfect for this as sign language is a mixture of both gestures and facial movements and the library can track both. however, it is a heavy library, and it may not be feasible to run an end to end application using it.

challenges


  • lack of good data available

  • model is not easy to train and requires significant resouces on scaling

  • requires regular feedback from someone with hearing or speaking impairments for improvements to ensure application is actually useful

key takeaways

  • developed an understanding of problems that might not directly affect me

  • learned computer vision from this project

  • learned how to apply lstm neural networks in the field of computer vision
